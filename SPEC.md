### 이벤트 분류 규칙
- **신규**: 이벤트 셀에 배경색 있음 (`#FFEAD1DC` 또는 `#FFFCE5CD`)
- **복각**: 텍스트에 `(복각)` 포함
- **상설**: 텍스트에 `(상설)` 포함 → 종료일 없음

### 픽업 분리 규칙
- `\n` 구분 → 별도 행 (개별 픽업 또는 배너)
- `,` 구분 → 동일 행 유지 (듀오 픽업, 속성1에 그대로 유지)

### 캠페인 분리 규칙
- `/` 구분 → 1주씩 순서대로 별도 행
- `x` → 해당 주차 해당 없음, 건너뜀

### 이벤트 셀 `\n` 처리
- 첫 번째 줄 → 속성1 (이벤트명)
- 이후 줄 → 속성3 (배포캐릭터)

## 총력전 분류 로직 (완료)

### 종류 판별
| 조건 | 종류 |
|------|------|
| 셀 텍스트 prefix `제약해제` | 제약해제결전 |
| 셀 배경색 있음 (ARMOR_FILL_MAP) | 대결전 |
| 그 외 | 총력전 |

### 장갑타입 판별
- **총력전**: 보스명 고정 테이블 (`RAID_BOSS_ARMOR`)
- **대결전**: 배경색 → 전체 방어타입 (S01~S11), 일섭 일정 매칭 (S12~) → 토먼트/그외 분리
- **제약해제결전**: 글씨색 (`ARMOR_COLOR_MAP`)

보스별 장갑타입:
```
비나/헤세드/호드/와카모 = 중장갑
예로니무스/카이텐/게부라/그레고리오 = 경장갑
고즈/시로쿠로/페로로질라 = 특수장갑
쿠로카게/예소드 = 탄력장갑
```

### 시즌 번호 (속성5)
- **총력전**: 일섭 날짜 ±15일 내 `RA_SCHEDULE` 매칭 → 시즌 라벨 (B, S00~S88)
  - 92건 중 83건 매칭, **9건 미매칭** (아래 참조)
  - 일섭 날짜도 RA_SCHEDULE의 정확한 일섭 날짜로 보정됨
- **대결전**: 일섭 날짜 ±15일 내 `DA_SCHEDULE` 매칭 → 시즌 라벨 (B1, B2, S01~S33)

## 알려진 미완성/미해결 항목

### 1. 총력전 미매칭 9건
위 표 참조. 시즌 라벨 없이 기존 일섭 날짜로 출력 중.

### 2. 총력전 `\n` 뒷부분 처리 미확인
일부 총력전 셀에 `\n경탄 / 특` 같은 부대 정보가 포함됨.
현재 속성5 앞에 쉼표로 이어붙이고 있으나 실제 활용 방식 미정.

### 3. 업데이트 항목 분류 검토 필요
배경색 없고 `(복각)/(상설)` 없는 이벤트 셀 내용을 일괄 `업데이트`로 분류 중.
실제로는 별도 종류가 필요한 항목이 섞여 있을 수 있음:
- 전술대항전, 합동화력연습 등 반복 컨텐츠
- 미니스토리

### 4. 한섭 날짜 없는 제약해제결전
일부 `제약해제결전` 항목이 한섭 날짜 없이 일섭 날짜만 존재 (20건).
스프레드시트에 한섭 날짜 미입력 상태. 현재 한섭 컬럼 빈칸으로 출력.

### 5. 마지막 항목들의 종료일 없음
스프레드시트 마지막 부근 항목들은 다음 항목이 없어 종료일 계산 불가.
이벤트/픽업 기준 마지막 항목들이 종료일 null 상태.

### 6. 픽업 종료일 불일치 (1건 확인)
2023-11-14 날짜에 히후미(수영복) 픽업의 종료일이 다른 픽업(나츠,마리/카즈사/레이사)과 다름.
- 나츠,마리 / 카즈사 / 레이사: 종료 2023-11-21
- 히후미(수영복): 종료 2023-11-28
원인: 스프레드시트에서 히후미 행이 2023-11-21 픽업 항목보다 뒤에 위치.
과거 데이터라 실사용 영향 없음.
